# Roadmap

## Current Phase: MVP — Validate Core Demand

**Goal:** Prove that sellers will list and buyers will transact — without WhatsApp as a fallback.
**Timeline:** Now → 30 days post-launch

---

## Phase 1 — MVP (Current)

### Buyer
- [x] Registration and authentication
- [x] GPS map discovery (stores, Market Squares, radius, filters)
- [x] Store and product browsing (media, reels)
- [x] Seller messaging
- [x] Paystack checkout
- [x] Follow stores + activity wall
- [x] Dassa AI shopping assistant

### Seller
- [x] Storefront setup (logo, description, hours, location)
- [x] Product listing with media upload
- [x] AI listing generator (photo → title, description, social captions)
- [x] Messaging inbox
- [x] Online/offline status
- [ ] Seller dashboard (orders, basic stats) — *in progress*

### Platform
- [x] Market Squares (map zones)
- [x] Stores ↔ Squares toggle
- [x] Deep-link store from product page
- [x] Verified badge + premium tier
- [x] Feed (fresh drops + reels)
- [ ] Admin panel (user + listing management) — *in progress*

---

## Phase 2 — Growth (Post-Validation)

**Goal:** Increase retention and transaction volume once core model is proven.
**Trigger:** 30 active sellers + 60% buyer conversion rate sustained for 2 weeks.

### Buyer
- [ ] Order tracking (integrated into Dassa)
- [ ] Product review and rating system
- [ ] Buyer notifications (new products, deals, messages)
- [ ] Saved stores and wishlists

### Seller
- [ ] Seller analytics dashboard (views, follows, sales, conversion)
- [ ] Bulk product upload
- [ ] Deal scheduling (set deal start/end time)
- [ ] Seller performance insights (AI-powered)
- [ ] Seller pin self-service (drag to correct location)

### Platform
- [ ] Push notifications (web + PWA)
- [ ] Category filter on map
- [ ] Promoted listings (paid visibility)
- [ ] Square events and announcements
- [ ] Affiliate/referral programme

---

## Phase 3 — Scale

**Goal:** Expand to more cities and categories. Introduce logistics and automation.
**Trigger:** Consistent monthly GMV growth + seller retention above 70%.

- [ ] Multi-city expansion (Abuja, Port Harcourt, Kano)
- [ ] Logistics and delivery integration
- [ ] Native mobile app (iOS + Android)
- [ ] Multi-currency (for diaspora buyers)
- [ ] B2B wholesale listings
- [ ] Seller subscription tiers
- [ ] Advanced AI — personalised buyer feed, price intelligence, demand forecasting

---

## What Moves Us Between Phases

| Condition | Action |
|---|---|
| Sellers stop listing after onboarding | Fix onboarding friction before Phase 2 |
| Buyers message but don't buy | Fix checkout trust/friction before Phase 2 |
| Dassa resolution rate below 50% | Retrain before scaling AI features |
| Admin workload unsustainable | Prioritise moderation tools above growth features |
