## Role Status Key

| Tag | Meaning |
|---|---|
| Core | Full member — paid, responsible, access per role |
| Advisor | Non-executive — feedback and guidance only |
| Junior / Probation | Performance reviewed each sprint — continuation not guaranteed |
| NJFP Fellow | National Youth Fellowship Programme — supervised, time-bound |
| Volunteer | Contribution-based — no guaranteed tasks or compensation |
| Conditional Performance | Continued engagement conditional on measurable output |
| Feedback Only | Advisory input only — no operational authority or system access |
